可执行的代码：
runA.py,runB.py,runC.py,运行利用weight文件夹下的.h5文件作为权重输入，并加载data中的图片，生成result中的图片；
DataMake_2A.py,DataMake_2B.py,DataMake_2C.py,运行后把result文件夹下的相应图片大小微调成要求的大小
其他代码由于没有数据，无法在其他电脑上运行。
以A为例，他们的功能介绍如下：
1.DataMakeA.py：用于制造大小和A.png相同的灰度图数据，分为两部分，分开执行。
第一部分为生成带噪声数据的图片，第二部分为更改图片大小为响应大小；
2.my_train_cnn.py：用于训练数据。输入之前制造的原图数据和被噪声干扰的数据，
调整卷积神经网络的参数使得带噪声图片经过卷积后能和原图差异缩小。
